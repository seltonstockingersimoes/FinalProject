My experience with VS Code:

I spent a lot of time working on Visual Studio Code because of my projects. I wouldn't say it's my favorite tool (That would be Photoshop.) but it is very effective.

One thing I didn't fully appreciate at first was how much extensions can completely change your experience, I just used the normal VS Code, I added a Color Highlight Extension and it really helped me with website design.

Tips and Tricks

- VS Code has a Zen mode, if you press Ctrl + K Z it will enter a full screen that will disable your notifications.

- By pressing Ctrl + P, you will instantly open any file.

- Be sure to always enable Auto Save and Format on Save, it saves you a lot of time and you won't worry about your code being lost.
