Semantic elements improve the structure of your code:

Semantic HTML elements are tags that describe the meaning and purpose of the content they contain Instead of using those generic tags for everything, you can such proper elements like: header, nav. and footer, it gives the developer and the browser a better understanding of the structure. Big projects (like this one) really need a good organized structure.

Beyond reading, the semantic elements also improve the accessbility and search optimization If you are a screen reader, you can jump between the sections, headers or navigations properly and for the person listening it will be very good. Always be sure to have a good structure.
